# @paycash/Paycash

## Install

```bash
npm install --save @paycash/paycash
```

## Usage

```
<script src="https://unpkg.com/@paycash/paycash/dist/paycash.js"></script>
<div class="paybutton"
  to="bitcoincash:qznnqftcnq34vprgvhhnmr9yukd9deprl5dptmkh4f"
  amount="5"
  currency="USD"
  animation="invert"
  text="Donate $5"
  hover-text="Send BCH"
  success-text="Thank you for supporting us"
></div>
```
