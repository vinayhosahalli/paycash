# @PacCash/react

> Paycash Is React Widget Helps To Accept BCH In Your Website


## Install

```bash
npm install --save @paycash/react
```

## Usage

```tsx
import React from 'react'

import { PayCash } from '@paycash/react'

<PayCash to={address} />
```
